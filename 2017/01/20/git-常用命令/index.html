git 常用命令

####常用命令
cd 进入目录 mkdir 创建目录
pwd 显示当前目录 git branch 查看当前分支
git branch 分支名 创建分支 git checkout 分支名 切换分支
git checkout -b 分支名 创建分支，然后切换到分支 git branch -d 分支名 删除分支
git checkout – 文件名 把文件在工作区的修改全部撤销(修改后还没有被放到暂存区) git reset HEAD 文件名 把暂存区的修改撤销掉，重新放回工作区(只是添加到了暂存区，还没有提交)
git add . 添加所有文件 git commit -m “提交描述” 提交记录
git push origin gh-pages 更新仓库 git status 查看修改过的文件
git diff 文件名 查看具体修改的内容 git log | git reflog 查看提交过的日志
git reset –hard HEAD HEAD代表当前版本 HEAD^代表上个版本 git reset –hard commit id 回退到某个版本
